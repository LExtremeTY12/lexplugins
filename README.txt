LEX Store — Static Website
==========================

This version does not need npm, React, Vite, or installation.

How to open:
1. Extract the ZIP file.
2. Open index.html in your browser.

What is included:
- index.html
- styles.css
- script.js
- assets/avatarHead.png
- assets/minecraft-background.png

Language:
- The website automatically detects the visitor's browser language.
- Portuguese browsers see Portuguese.
- All other browsers see English by default.
- Visitors can also switch manually using the EN/PT buttons in the header.

How to edit products:
1. Open script.js.
2. Find the PRODUCTS list.
3. Add, remove, or change products there.

Button options:
- actionType: "discord" sends the customer to the Discord invite.
- actionType: "link" sends the customer to the product URL.
- type: "free" makes the button use a download label.
